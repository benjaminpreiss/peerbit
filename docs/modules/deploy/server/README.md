# Setting up a server node
## Testing locally

To run a node (and relay) locally you can use the CLI

```sh
npm install -g @peerbit/server
```

```sh
peerbit start
```

Run 
```sh
peerbit --help
```
for more info

## Configure a server
To configure an existing server to run a reachable node, [read this](./custom.md).
